# Security Policy

Hedron is maintained by Glorian Labs. We appreciate responsible disclosure.

## Reporting a Vulnerability

Please report security issues privately to:

- `security@glorianlabs.com`

Include:

- Affected component(s) and version/commit
- Reproduction steps or proof of concept
- Severity/impact assessment
- Suggested mitigation (if available)

## Disclosure Process

1. We acknowledge reports as quickly as possible
2. We validate and triage impact
3. We prepare and test a fix
4. We coordinate disclosure timing with the reporter

Please avoid public disclosure until a fix is available or coordinated.
